# SwiftLintCISample [![Circle CI](https://circleci.com/gh/oppai/SwiftLintCISample/tree/master.svg?style=svg)](https://circleci.com/gh/oppai/SwiftLintCISample/tree/master)
SwiftLint CI Test Sample for CircleCI

### Build

```shell
$ bundle install
$ bundle exec pod install
```

### Install SwiftLint

```shell
$ brew install swiftlint
```
